LINEWIDTH

Syntax: SET LINEWIDTH n
Default value: LINEWIDTH = 1

LINEWIDTH controls the line width of the axes, the data curve, and the plotting symbols drawn when the GRAPH command is entered.

The units of LINEWIDTH are 1/4 pts, where a pt is 1/72 of an inch. For example, a line width of 1 gives 1/288 inch wide lines, while a line width of 4 gives 1/72 inch wide lines.

The parameter n must be a scalar.

SET LINEWIDTH is a shorthand way to set CURVELINEWIDTH and PLOTSYMBOLLINEWIDTH.

GET LINEWIDTH is ambiguous, and will produce a syntax error. Use GET CURVELINEWIDTH or GET PLOTSYMBOLLINEWIDTH instead.

Other line widths that can be set include XAXISLINEWIDTH, YAXISLINEWIDTH, and POLARAXISLINEWIDTH.
