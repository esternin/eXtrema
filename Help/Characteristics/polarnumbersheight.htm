POLARNUMBERSHEIGHT

Syntax: SET POLARNUMBERSHEIGHT n
SET %POLARNUMBERSHEIGHT n
GET POLARNUMBERSHEIGHT { n }
GET %POLARNUMBERSHEIGHT { n }
Default: %POLARNUMBERSHEIGHT = 3

POLARNUMBERSHEIGHT controls the size of the numbers labeling the polar axis. There are two forms:

 POLARNUMBERSHEIGHT → the height, in world coordinate units, of the numbers labeling the polar axis
%POLARNUMBERSHEIGHT → the height, expressed as a percentage of the height of the graphics sub-window, that is, POLARNUMBERSHEIGHT = %POLARNUMBERSHEIGHT * (YUPPERWINDOW- YLOWERWINDOW)/100.
